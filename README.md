        # kafka — Продьюсер: send, acks, idempotence

        Homework template for the Vibe Learn lesson **l3_producer_basics** (Продьюсер: send, acks, idempotence).

        ## Task

        Producer benchmark на Go: измерь throughput и p99 latency при трёх конфигурациях acks
(acks=0, acks=1, acks=all). Запусти локальный Kafka через Docker Compose, отправь по
10 000 сообщений в каждом режиме, зафиксируй время. CI проверяет: (1) acks=0 быстрее acks=1
быстрее acks=all (по суммарному времени); (2) ни одна из трёх прогонов не завершилась с ошибками;
(3) общее число залогированных ack совпадает с числом отправленных сообщений для acks=1 и acks=all.

        ## How to work

        1. The Vibe Learn platform creates a copy of this repo in your GitHub account when you click «Начать домашку» on the lesson.
        2. Clone it locally, complete the TODOs in `main.go`, and push.
        3. CI (`.github/workflows/ci.yml`) runs the tests on every push. The Vibe Learn platform listens to the result via webhook and unlocks the lesson when CI passes.

        ## Local prerequisites

        - Go 1.22+
        - Docker + docker-compose (for the local Kafka cluster, when the lesson requires it)

        ## Running

        ```bash
        go test ./...
        ```
